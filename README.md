# Heurística para encontrar árvores de Steiner usando o Surface Evolver.

### Instruções

1. Faça o download dos arquivos usando usando o botão **Clone or download**.
2. Rode o arquivo **main.py** usando o Python.
3. O comando **python main.py arq -m exemplos/cruzamento1.txt** roda o algoritmo de Prim no arquivo cruzamento1.txt
4. O comando **python main.py arq -s exemplos/cruzamento1.txt** roda o algoritmo Showcross no arquivo cruzamento1.txt
5. O comando **python main.py arq -n exemplos/cruzamento1.txt** roda o algoritmo Nocross no arquivo cruzamento1.txt
6. O comando **python main.py gera exemplos/pontos 100 1** gera 100 pontos com pesos no arquivo *pontos.txt* na pasta *exemplos*
7. O comando **python main.py gera exemplos/pontos 100 0** gera 100 pontos sem pesos no arquivo *pontos.txt* na pasta *exemplos*
